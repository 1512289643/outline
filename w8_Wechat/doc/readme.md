## day8-2

* 防抖与节流
    > 为了优化页面性能
    * 防抖：只执行最后一次，忽略之前的所有操作
        * 百度搜索建议
        * js动画
    * 节流：执行第一次，忽略后面所有操作
        * 滚动加载
